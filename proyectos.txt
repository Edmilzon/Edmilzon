user_blog_backend : Es una API REST con 4 entidades relacionadas. Permite CRUD sobre usuarios, roles, blogs y artículos. Cada usuario puede tener un blog (1 a 1), múltiples artículos y roles asignados, y los artículos pertenecen a un usuario y un blog.
Técnicamente: Spring Boot 3.2.2 + JPA/Hibernate + PostgreSQL, con endpoints REST en /user, /role, /blog y /article. La estructura de capas (Controllers → Services → Models/DTOs) está armada, pero la capa de repositorio y la lógica de negocio con base de datos están pendientes de implementar.

Control_voz_gesto_java: Una aplicación de escritorio Java Swing que organiza objetos automáticamente en una cuadrícula con 6 patrones distintos (L, U, cuadrado, cruz, líneas horizontales/verticales separadas). Una vez organizados, podés mover los objetos por la cuadrícula usando teclado (WASD), voz en español (arriba, abajo, izquierda, derecha) o gestos de la mano detectados por cámara web con OpenCV.

Detectar_videos_imagen_java: l proyecto es una aplicación Java de escritorio que usa OpenCV para visión por computadora. Tiene dos modos:
Paquete 1 (CLI) — menú interactivo en consola para:
- Detectar objetos en una imagen (contornos con umbral Otsu)
- Detectar movimiento entre dos imágenes (diferencia de píxeles)
- Detectar objetos en un video (por rango de color HSV + circularidad)
- Detectar movimiento en un video (diferencia entre frames consecutivos)
Paquete 2 (GUI + Cámara) — interfaz Swing para:
- Grabación de video desde webcam con detección de rostros en tiempo real (Haar cascades)
- Capturar fotos desde cámara
- Comparar imágenes (template matching, momentos de Hu)
- Umbralizado, desenfoque, conteo y extracción de rostros
- Reconocimiento de voz con JSAPI para navegación por comandos de voz
En resumen: es un toolbox de visión artificial con OpenCV, manejable desde terminal o interfaz gráfica.
